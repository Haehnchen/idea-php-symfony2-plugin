Add Symfony Route attribute to controller action

Adds a #[Route] attribute to a public method in a Symfony controller class.

The intention is available when:

  • The method is public and non-static
  • The method doesn't already have a Route attribute
  • The class is a controller (extends AbstractController, has #[AsController], or has Route attributes)

The route path and name are automatically generated based on the controller and method name:

  • App\Controller\ProductController::index → #[Route('/product', name: 'app_product_index')]
  • App\Controller\Admin\UserController::edit → #[Route('/admin/user/edit', name: 'app_admin_user_edit')]
