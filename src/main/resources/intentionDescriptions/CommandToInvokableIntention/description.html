Migrate Symfony Command to invokable style (Symfony 7.3+)

Converts a traditional Symfony Command with execute() method to the new invokable command style with __invoke() method using SymfonyStyle and input attributes.

This migration:

  • Renames execute() to __invoke()
  • Replaces InputInterface and OutputInterface with SymfonyStyle $io
  • Converts addArgument() and addOption() calls to #[Argument] and #[Option] attributes
  • Updates method body to use direct parameter access instead of $input->getArgument()
  • Removes configure() method if it only contains argument/option definitions
