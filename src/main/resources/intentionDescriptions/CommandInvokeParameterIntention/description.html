Add parameter to invokable Symfony Command __invoke method

Adds a parameter to the __invoke() method of an invokable Symfony Command.

Available parameters:

  • InputInterface - Access to command input
  • OutputInterface - Access to command output
  • Cursor - Control cursor position in terminal
  • SymfonyStyle - Styled input/output helper
  • Application - Access to console application

The intention is only available when the class does not extend Command and has an __invoke() method.
