Detects usage of deprecated PHP methods in Twig templates.

This inspection analyzes Twig templates and identifies when variables or method calls reference PHP methods that have been marked as deprecated using either @deprecated annotation or #[Deprecated] attribute.

The inspection checks for:

  • Deprecated methods called in Twig templates
  • Methods marked with @deprecated or #[Deprecated] PHPDoc/attribute

Example Twig template: 


{# If User::getUsername() is deprecated, this will be highlighted #}
{{ user.username }}

{# If ProductService::getFeaturedProducts() is deprecated, this will be highlighted #}
{% for product in products.getFeaturedProducts() %}
    {{ product.name }}
{% endfor %}
