Inspection that detects missing Twig templates when using PHP attributes or PHPDoc annotations.

This inspection validates template references in PHP attributes (like #[Template]) and PHPDoc annotations (like @Template) to ensure the referenced template files actually exist in the project.

The inspection supports:

  • PHP 8 attributes: #[Template("template.html.twig")]
  • PHPDoc annotations: @Template("template.html.twig")
  • Auto-detection of template names for empty attributes/annotations based on controller and method names
  • Both .html.twig and other template extensions

When a missing template is detected, the inspection provides quick fixes:

  • Create the missing template file
  • Suggest possible typos in the template name
Example: 
use Symfony\Bundle\FrameworkBundle\Controller\AbstractController;
use Symfony\Component\HttpFoundation\Response;
use Symfony\Component\Routing\Annotation\Route;

class BlogController extends AbstractController
{
    #[Route("/blog/{slug}")]
    #[Template("blog/show.html.twig")] // Missing template
    public function show(string $slug): Response
    {
        return [];
    }
}
