Inspection that detects deprecated controller actions referenced in YAML routing configurations.

This inspection analyzes route definitions in YAML routing files (routes.yaml, routing.yaml, etc.) and identifies when the referenced controller actions (methods or controller classes) have been marked as deprecated.

The inspection checks for:

  • Deprecated controller methods or classes using @deprecated or #[Deprecated]
  • Controller actions that should no longer be used in new code
Example YAML route configuration: 

# config/routes.yaml
blog_show:
    path: /blog/{slug}
    controller: App\Controller\BlogController::show  # Will be highlighted if deprecated
Or using the legacy _controller key: 

# config/routes.yaml
blog_show:
    path: /blog/{slug}
    defaults:
        _controller: App\Controller\BlogController::show  # Will be highlighted if deprecated

If the BlogController::show method or the BlogController class is marked as deprecated, this inspection will highlight the route definition and warn about the deprecation.
