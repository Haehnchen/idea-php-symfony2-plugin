Detects references to Symfony routes that cannot be found in routing configuration.

This inspection analyzes Twig templates and identifies when route names used in path() and url() functions don't correspond to any routes defined in the Symfony routing configuration.

The inspection checks for missing routes in:

  • {{ path('route_name') }}
  • {{ url('route_name') }}
  • {{ path('route_name', {'param': 'value'}) }}
  • {{ url('route_name', {'param': 'value'}) }}

Example Twig template: 


{# This will be highlighted if 'homepage' route doesn't exist #}
Home

{# This will be highlighted if 'product_detail' route doesn't exist #}
{{ path('product_detail', {'id': product.id}) }}

{# This will be highlighted if 'contact_form' route doesn't exist #}
{{ url('contact_form') }}

{# This will be highlighted if 'admin_dashboard' route doesn't exist #}
{{ path('admin_dashboard') }}

When a missing route is detected, the inspection provides quick fixes:

  • Suggest possible typo corrections in the route name
  • Help identify the correct route name from available routes
