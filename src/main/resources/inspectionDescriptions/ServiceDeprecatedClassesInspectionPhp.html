Service or class referenced in PHP code is deprecated

This inspection detects deprecated services and classes when they are used in PHP code. It identifies services or classes that have been marked as deprecated and should be replaced with newer alternatives.

The inspection checks for:

  • Service classes with @deprecated PHPDoc annotation
  • Classes marked with #[Deprecated] attribute
  • Services explicitly marked as deprecated in Symfony's service definition

Example PHP code that would trigger this inspection: 


// This will trigger inspection if class is deprecated
use App\Legacy\OldService;

class MyService
{
    public function __construct(
        OldService $oldService,  // OldService has @deprecated annotation
        #[Required] DeprecatedService $depService
    ) {
        // ...
    }
}

// This will trigger inspection for service ID
public function someMethod(ContainerInterface $container): void
{
    $service = $container->get('deprecated_service');
    // This service is marked as deprecated in Symfony
}

// This will trigger inspection for dependency injection
#[Route('/example')]
class ExampleController extends AbstractController
{
    public function __construct(
        private OldLegacyService $legacyService  // Deprecated service
    ) {}
}
