Detect deprecated Twig extensions including tags, filters, and functions

Reports the usage of deprecated Twig extensions:

  • Tags: Deprecated TokenParser classes via @deprecated annotation, #[Deprecated] or DeprecatedCallableInfo::triggerDeprecation attribute (e.g., {% spaceless %})
  • Filters: Deprecated via "deprecated" or "deprecation_info" options in Twig extension (e.g., {{ value|deprecated_filter }})
  • Functions: Deprecated via "deprecated" or "deprecation_info" options in Twig extension (e.g., {{ deprecated_function() }})

This inspection helps identify and migrate away from deprecated Twig functionality to ensure compatibility with future Twig versions.
