Detects deprecated Symfony route configuration settings in YAML routing files.

This inspection analyzes YAML routing files and identifies deprecated route configuration options that should be replaced with their modern equivalents in newer Symfony versions.

The inspection checks for deprecated settings in:

  • pattern - Should be replaced with path
  • _method requirement - Should use HTTP method constraints directly
  • _scheme requirement - Should use scheme constraints directly

Example deprecated YAML configuration: 


# config/routes.yaml
blog_show:
    # DEPRECATED: 'pattern' should be 'path'
    pattern: /blog/{slug}
    controller: App\Controller\BlogController::show
    requirements:
        # DEPRECATED: '_method' should use 'methods' directly
        _method: GET
        # DEPRECATED: '_scheme' should use 'schemes' directly
        _scheme: https

Example modern YAML configuration: 


# config/routes.yaml
blog_show:
    # MODERN: Use 'path' instead of 'pattern'
    path: /blog/{slug}
    controller: App\Controller\BlogController::show
    # MODERN: Use 'methods' and 'schemes' directly
    methods: [GET]
    schemes: [https]
