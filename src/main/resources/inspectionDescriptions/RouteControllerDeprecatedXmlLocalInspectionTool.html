Inspection that detects deprecated controller actions referenced in XML routing configurations.

This inspection analyzes route definitions in XML routing files (routes.xml, routing.xml, etc.) and identifies when the referenced controller actions (methods or controller classes) have been marked as deprecated.

The inspection checks for:

  • Deprecated controller methods or classes using @deprecated or #[Deprecated]
  • Controller actions that should no longer be used in new code
Example XML route configuration: 

<routes>
    <route id="blog_show" path="/blog/{slug}">
        <default key="_controller">
            App\Controller\BlogController::show  <!-- Will be highlighted if deprecated -->
        </default>
    </route>
</routes>

If the BlogController::show method or the BlogController class is marked as deprecated, this inspection will highlight the route definition and warn about the deprecation.
