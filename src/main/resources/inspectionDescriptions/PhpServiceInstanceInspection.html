Provided instance of service id does not match argument

This inspection detects service references in PHP service configuration whose resolved class is not an instance of the expected constructor parameter type.

The inspection checks for:

  • Service references in PHP array-style 'arguments' using service(), ref(), or raw '@service_id'
  • Service references in PHP fluent ->args([...]) using service() or ref()

Example PHP code that would trigger this inspection: 


// Array-style service configuration
return [
    MyService::class => [
        'arguments' => [
            service('wrong_type_service'), // triggers inspection - wrong type
            ref('wrong_type_service'),     // triggers inspection - wrong type
            '@wrong_type_service',         // triggers inspection - wrong type
        ],
    ],
];

// Fluent-style service configuration
$services->set(MyService::class)
    ->args([
        service('wrong_type_service'), // triggers inspection - wrong type
    ]);
