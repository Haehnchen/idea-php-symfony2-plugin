Detects references to Twig templates that cannot be found in configured template paths.

This inspection analyzes Twig templates and identifies when template references (includes, embeds, extends, etc.) point to template files that don't exist in any of the configured template directories.

The inspection checks for missing templates in:

  • {% include 'template.html.twig' %}
  • {% embed 'template.html.twig' %}
  • {% extends 'template.html.twig' %}
  • {{ include('template.html.twig') }}
  • {{ source('template.html.twig') }}

Example Twig template: 


{# This will be highlighted if 'header.html.twig' doesn't exist #}
{% include 'header.html.twig' %}

{# This will be highlighted if 'forms/contact.html.twig' doesn't exist #}
{% embed 'forms/contact.html.twig' %}
    {% block content %}
        Contact form content
    {% endblock %}
{% endembed %}

{# This will be highlighted if 'base.html.twig' doesn't exist #}
{% extends 'base.html.twig' %}

When a missing template is detected, the inspection provides quick fixes:

  • Create the missing template file with the appropriate directory structure
  • Suggest possible typo corrections in the template name
