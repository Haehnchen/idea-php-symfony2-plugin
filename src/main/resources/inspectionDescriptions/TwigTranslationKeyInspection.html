Translation key is not found in any translation file

This inspection detects translation keys in Twig templates that don't exist in any translation file. It validates that trans, transchoice, and filter translation calls reference valid translation keys.

The inspection checks for:

  • Translation keys used in {{ trans() }} or {{ transchoice() }} that don't exist
  • Translation keys in {% trans %} blocks that are missing from translation files
  • Translation keys used with |trans filter that are not defined

Example Twig template that would trigger this inspection: 


{# This will trigger inspection if key doesn't exist #}
{{ 'missing.translation.key'|trans }}

{# This will trigger inspection if key doesn't exist #}
{% trans with {'%name%': 'John'} from 'messages' %}hello.welcome{% endtrans %}

{# This will trigger inspection if key doesn't exist #}
{{ transchoice(count, 'item.none', 'item.some') }}

{# This is valid - key exists in translations/messages.en.yml #}
{{ 'homepage.welcome'|trans }}

A quick fix is available to automatically create the missing translation key in the appropriate translation file.
