Detects references to Twig templates in PHP code that cannot be found in configured template paths.

This inspection analyzes PHP code and identifies when template references in Symfony/Twig methods point to template files that don't exist in any of the configured template directories.

The inspection checks for missing templates in:

  • $this->render('template.html.twig')
  • $this->renderView('template.html.twig')
  • $twig->render('template.html.twig')
  • $twig->loadTemplate('template.html.twig')
  • $engine->render('template.html.twig')
  • $this->stream('template.html.twig')

Example PHP controller: 


class BlogController extends AbstractController
{
    // This will be highlighted if 'blog/index.html.twig' doesn't exist
    public function index(): Response
    {
        return $this->render('blog/index.html.twig', [
            'posts' => $this->getDoctrine()->getRepository(Post::class)->findAll()
        ]);
    }

    // This will be highlighted if 'blog/show.html.twig' doesn't exist
    public function show(Post $post): Response
    {
        return $this->render('blog/show.html.twig', [
            'post' => $post
        ]);
    }
}

Example PHP service with Twig: 


class EmailService
{
    private $twig;

    public function __construct(Environment $twig)
    {
        $this->twig = $twig;
    }

    // This will be highlighted if 'emails/welcome.html.twig' doesn't exist
    public function sendWelcomeEmail(User $user): void
    {
        $content = $this->twig->render('emails/welcome.html.twig', [
            'user' => $user
        ]);

        // send email logic...
    }
}

When a missing template is detected, the inspection provides quick fixes:

  • Create the missing template file with the appropriate directory structure
  • Suggest possible typo corrections in the template name
