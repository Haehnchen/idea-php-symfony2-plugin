Consider adding an int return type to the __invoke() method in Symfony Commands

Symfony 7.3+ invokable commands using the #[AsCommand] attribute should declare int as the return type for the exit code.

Valid return values are:

  • Command::SUCCESS (0) - Command completed successfully
  • Command::FAILURE (1) - Command failed
  • Command::INVALID (2) - Invalid input

See Symfony Console documentation for more information.
